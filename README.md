# Features

- create-react-app
- The source code was refactored by Typescript
- react hook
- json-server for mock API server

# How to use

- Install packages

```shell
npm run install
```
- Start application

```shell
# Start json-server for mock API.
npm run server

# Start application.
npm run start
```

- Build

```shell
npm run build
```
